7. Lessons Learned

  1. Feature family 편중이 signal exploration의 진짜 병목이었다. Primitive whitelist를 확장하니 LLM이 자연스럽게 새 spec을 생성, top 10을 모두 새 family가 점령.
  2. Direction flip은 critical: ask_depth_concentration, obi_ex_bbo, zscore(trade_imbalance_signed) 3개는 contra-directional. 잘못된 direction으로 실험하면 "signal이 작동 안 함"으로 오진.
  3. Compound formulas win: top 3 모두 zscore(Block C primitive) * regime_filter 구조. 단순 zscore는 10위권, compound 첨가로 expectancy 30% 상승.
  4. Regime filter의 위력: rolling_realized_vol < 30 (저변동 regime)에서만 진입 → 신호 fidelity 극대화, sample size 감소와 trade-off.
  5. Horizon mismatch 주의: 다중 horizon curve 자동화가 feature별 optimal horizon 발견에 결정적. Default h=1 고정 실험은 새 family의 real edge를 놓침.
  6. Duplicate-reject으로 인한 자동 종료: signal-evaluator가 prior_iterations_index.md를 엄격히 비교하여 유사 spec reject. 22 iter 이후 탐색공간 고갈 신호. 다음 run에선 duplicate 판정 완화 또는 강제 다양성 주입 필요.
